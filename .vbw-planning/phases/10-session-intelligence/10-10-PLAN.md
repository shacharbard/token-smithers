GREEN — src/token_sieve/intent/detector.py:
  ```python
  class IntentDetector:
      def __init__(self, window_size: int = 5, rules=None) -> None:
          from .rules import RULES
          self._window_size = window_size
          self._rules = rules if rules is not None else RULES
          self._window: deque[str] = deque(maxlen=window_size)
      def observe(self, tool_name: str) -> None:
          self._window.append(tool_name)
      def current_mode(self) -> str:
          window = list(self._window)
          for pattern, mode in self._rules:
              if self._matches(window, pattern):
                  return mode
          return "unknown"
      @staticmethod
      def _matches(window, pattern) -> bool:
          # Contiguous suffix match: window ends with pattern
          if len(window) < len(pattern): return False
          return tuple(window[-len(pattern):]) == tuple(pattern)
  ```
Design decision: match = contiguous SUFFIX (window ends with pattern). This matches the CONTEXT D3 sketches where `(Read × 4)` means "the last 4 calls were all Read". Document this in the module docstring.
Commit GREEN.
