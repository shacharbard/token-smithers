GREEN — imports_graph.py:
  ```python
  def blast_radius(graph: ImportsGraph, changed: set[str]) -> set[str]:
      seen = set(changed)
      queue = deque(changed)
      while queue:
          node = queue.popleft()
          for importer in graph.importers_of(node):
              if importer not in seen:
                  seen.add(importer)
                  queue.append(importer)
      return seen
  ```
Commit GREEN.
