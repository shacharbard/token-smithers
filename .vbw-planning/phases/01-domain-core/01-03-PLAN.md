## Objective

Build a CLI test harness that pipes text through the CompressionPipeline and reports token savings (original count, compressed count, savings ratio). Add a zero-dependency enforcement test that verifies the domain core has no external imports. This plan completes Phase 1's success criteria for CLI verification and dependency purity.

### Task 2: CLI main implementation (TDD: GREEN)
**Files:** `src/token_sieve/cli/main.py`

1. GREEN: Implement `main.py`:
   - `def create_pipeline() -> CompressionPipeline` — creates pipeline with CharEstimateCounter, registers a PassthroughStrategy (Phase 1 default, compresses nothing)
   - `def run(input_text: str, pipeline: CompressionPipeline, counter: TokenCounter) -> tuple[str, dict]` — processes text, returns (output, stats_dict)
   - `def main(argv: list[str] | None = None) -> int` — CLI entry point: parse args, read input (stdin or file), call run(), print output to stdout, print stats to stderr
   - Stats format: `Original: {n} tokens | Compressed: {n} tokens | Savings: {ratio:.1%}`
2. Add `[project.scripts]` entry in `pyproject.toml`: `token-sieve = "token_sieve.cli.main:main"` (update existing pyproject.toml from Plan 1)
3. Confirm all CLI tests pass.

### Task 3: Zero-dependency enforcement test (TDD: RED then GREEN)
**Files:** `tests/unit/domain/test_zero_deps.py`

1. RED: Write `test_zero_deps.py`:
   - `test_domain_has_no_external_dependencies` — uses `subprocess.run()` to spawn a Python process that imports `token_sieve.domain` with a stripped sys.path (only stdlib + src/). Asserts no ImportError.
   - `test_domain_modules_import_only_stdlib` — use `importlib` and `inspect` to walk all domain module imports and assert none are third-party (check against a known stdlib module list or use `sys.stdlib_module_names` on Python 3.10+)
2. GREEN: These tests should pass immediately if the domain core is correctly implemented (no external deps). If any fail, it indicates an accidental dependency leak that must be fixed in the domain modules.
3. Confirm tests pass.
